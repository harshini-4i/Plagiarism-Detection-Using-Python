# Plagiarism Detection Using Python

A simple Python project that compares two text files and calculates their similarity using the `difflib` library.

## Features
- Compares two text files
- Calculates similarity percentage
- Easy to understand and use

## Files
- plagiarism detector.py
- demo1.txt
- demo2.txt

## How to Run

```bash
python "plagiarism detector.py"
```

## Sample Output

```
The plagiarized content is 60.61%
```
